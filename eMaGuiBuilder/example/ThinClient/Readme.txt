Beispiel für einen Ultra Thin Client
====================================

Anwendung starten:
- StartServer.bat aufrufen
- Im Browser folgende URL eingeben:
http://localhost/ThinClientApplet.html

Voraussetzung:
- Das Java Plug-in für JDK 1.4 muß installiert sein.
- Eine Personal Firewall zuvor deaktivieren!
- Wenn der GuiBuilder nicht unter "/guibuilder" installiert ist,
  dann in config.xml <Docbase> entsprechend anpassen.
