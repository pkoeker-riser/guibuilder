Version 0.9.7.5

  • Das Attribut autosize=[true|false] ist neu.
  • Validierung von Text-Eingaben in Tabelle korrigiert: Das Verhalten entspricht dem von GuiText.

Version 0.9.7.4 (4.1.2004)

Version 0.9.7.3 (16.11.2003)

  • Die Möglichkeit den GuiBuilder mit Plugins zu erweitern ist neu; vorerst nur für die Komponenten GuiText und GuiDate.
  • Umstellung von HashTable auf HashMap bei Container.

  • Es gibt bei der Tabelle zwei neue Methoden setSelectedRow und setSelectedRows
  • Binding von DataSet mit Tabelle korrigiert.
  • OnChange-Ereignisse werden jetzt auch von Tabellen-Spalten ausgelöst.
  • Neue Methode GuiTable#setItems(String columnName, JDataSet ds) für ComboBoxen in Tabellen; hier wird auch der Renderer richtig versorgt.
  • Wenn GuiDate oder GuiTime leer ist, wird bei getValue NULL statt einem Leerstring geliefert.
    Das ist eine allgemeines Problem von JTextField: Wenn NULL mit setText() zugewiesen wird, wird anschließend bei getText() ein Leerstring geliefert :-(
  • Der Controller eines GuiWindows darf seine Methoden auch vererben (getMethod statt getDeclaredMethod).
